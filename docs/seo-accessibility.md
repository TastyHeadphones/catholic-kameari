# SEO And Accessibility Guide

## SEO

- Use Yoast SEO for page titles, meta descriptions, Open Graph, and XML sitemap.
- Keep canonical URLs enabled.
- Submit sitemap to Google Search Console after launch.
- Use local church structured data from the child theme.
- Keep old post permalink structure `/%year%/%postname%/`.
- Import redirects from `migration/redirection-plugin-import.csv`.
- Compress large images before upload.
- Use descriptive Japanese titles for posts and pages.

## Recommended Page Metadata

- Home: `カトリック亀有教会 | Catholic Kameari Church, St. Francis of Assisi`
- Mass: `ミサのご案内 | カトリック亀有教会`
- First-time visitors: `初めての方へ | カトリック亀有教会`
- Access: `アクセス | カトリック亀有教会`
- Weddings: `結婚式 | カトリック亀有教会`
- Funerals and cemetery: `葬儀・共同墓地 | カトリック亀有教会`

## Accessibility

- Keep body text large enough for elderly visitors.
- Preserve strong color contrast.
- Use semantic headings.
- Make navigation keyboard accessible.
- Add alt text to every meaningful image.
- Do not rely on color alone for important notices.
- Avoid dense text blocks on mobile.
- Make phone numbers and Google Maps links easy to tap.
- Use the browser preview to test mobile menu and focus states.

## Performance

- Use server-level cache or LiteSpeed Cache depending on hosting.
- Avoid Elementor unless necessary.
- Keep plugin count low.
- Resize images before upload.
- Prefer WebP or optimized JPEG where supported.
- Use lazy loading for below-the-fold images.
